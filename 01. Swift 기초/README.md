## 기본 명명 규칙

> 변수, 상수, 함수, 메서드, 타입 등의 이름은 유니코드에서 지원하는 어떤 문자든 사용 가능하다 <br/>
<u>하지만</u> 스위프트의 예약어와 키워드, 해당 코드 범위내에서 미리 사용되는 기존 이름과 동일한 이름, 연산자로 사용되는 기호(+,-,*,/), 숫자로 시작하는 이름, 공백이 포함된 이름은 안된다
